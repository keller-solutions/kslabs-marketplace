# Keller Solutions Core

Core skills and utilities from Keller Solutions for Claude Code.

## Installation

```bash
/plugin install keller-solutions-core@keller-solutions-marketplace
```

## Available Skills

| Skill | Description |
|-------|-------------|
| skill-template | Template demonstrating proper SKILL.md structure |
| marketplace-ops | Programmatic marketplace operations for agents |
